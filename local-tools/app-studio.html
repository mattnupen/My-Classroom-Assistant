App Studio

Your AI builds these — describe a problem and it builds the tool.

How it works: each prompt starts with /teacher-app-builder — the skill that builds your tool and checks it can’t leak student data. Copy a prompt, paste it into Claude Cowork, answer a couple of short questions, and the new tool lands in this sidebar.
First time? Add the teacher-app-builder skill once before using these — install teacher-app-builder-skill-upload.zip (steps in docs/teacher-app-builder.md), or add it from Claude’s Settings → Capabilities. After that, /teacher-app-builder just works.
Why this is safe: every tool built this way is one offline HTML file — no internet, no cloud, no student data saved anywhere. The privacy line is the same one every built-in tool follows.
